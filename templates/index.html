Understanding Stock Analysis & Predictions

Technical Indicators

RSI (Relative Strength Index)

A momentum oscillator that measures the speed and magnitude of price movements:

  • Range: 0 to 100
  • Above 70: Potentially overbought condition
  • Below 30: Potentially oversold condition
  • 50: Neutral level, used as centerline

MACD (Moving Average Convergence Divergence)

Trend-following momentum indicator showing relationship between moving averages:

  • Positive MACD: Short-term momentum is upward
  • Negative MACD: Short-term momentum is downward
  • MACD crossing signal line: Potential trend change

Price Predictions

Our ML model generates predictions using multiple data points:

  • Next Day: Short-term prediction based on recent price action and indicators
  • Next Week: Medium-term forecast incorporating broader market trends
  • Next Month: Long-term projection using historical patterns and technical analysis

Note: All predictions are probability-based estimates and should not be used as sole trading decisions.

Blockchain Verification

Each prediction is secured using blockchain-style verification:

  • Unique hash generated for each prediction
  • Hash includes timestamp, stock symbol, and prediction values
  • Allows verification of when predictions were made
  • Ensures prediction integrity and transparency
